# MathFlow - AI Math Learning Platform

This project is a Proof of Concept for an educational mathematics platform assisted by AI. It features a modular architecture, focusing on pedagogical effectiveness and robust security.

## Core Features
- **Socratic AI Tutor:** An intelligent assistant that guides students without giving away direct answers.
- **Premium Learning Paths:** Curated educational content with secure payment flows.
- **Microservices Architecture:** Scalable backend, specialized AI service, and specialized frontends.
